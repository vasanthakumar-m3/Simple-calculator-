# 🧮 Simple Calculator

A simple and user-friendly calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is built to demonstrate fundamental programming concepts and a clean user interface. 0

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear Input
- ⚠️ Basic Error Handling
- 📱 Responsive & Easy-to-Use Interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Simple-calculator-
│
├── index.html

├── script.js
└── README.md
```

## ⚙️ Prerequisites

- Any modern web browser
- VS Code (Recommended)

## 🔧 Installation

### 1. Clone the repository

```bash
git clone https://github.com/vasanthakumar-m3/Simple-calculator-.git
```

### 2. Navigate to the project

```bash
cd Simple-calculator-
```

### 3. Run the application

Open the `index.html` file in your preferred web browser.
